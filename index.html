Current Version: 0.0.7

Repository addon

Instructions for adding this repo:

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://i-a-c.github.io/repo/ (Give it the name "Exodus Redux").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "Exodus Redux", and install repository.exodusredux-0.0.7.zip
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "Exodus Redux Repo"
  • Go into the "Video add-ons" section in the repo, and you'll find Exodus Redux

Enjoy!
